# Windows CMD

## CMD/PowerShell Commands

### Get serial number
`wmic bios get serialnumber`

### Get Battery Health
`powercfg /batteryreport`

## Placering af dump logs
**Memory dumps - mini dumps:** `C:\Windows\Minidump`
**Full dumps:** `C:\Windows\MEMORY.DMP `
